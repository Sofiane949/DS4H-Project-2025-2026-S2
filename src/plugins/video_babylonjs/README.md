## Installation

```bash
cd video_babylonjs
yarn install
yarn build                                      
```   

## Intégration WAM

Le plugin `video_babylonjs` est conçu pour être enregistré dans le dépôt `wam-community`. Il utilise l'extension **Video** pour recevoir les flux de pixels des autres plugins (comme Butterchurn ou ISF)
et les projeter sur des objets 3D dans un espace spatialisé.
